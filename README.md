## Getting Started

1. **Clone the Repository**

   ```bash
   git clone https://github.com/sohrb/bitpin-interview-task.git
   ```

2. **Install packages**

   ```bash
   pnpm i
   ```

   ```bash
   npm install
   ```

   ```bash
   yarn
   ```

3. **Run the project**

   ```bash
   pnpm dev
   ```

   ```bash
   npm run dev
   ```

   ```bash
   yarn dev
   ```
